# Jammming Final Project Codecademy

This is a playlist generator created in React.js and deployed with Surge. It uses the Spotify API and OAuth to search Spotify for Artists, Songs or Albums allowing the user to generate a new playlist and save to their Spotify account.


## Built With

* [React.js](https://reactjs.org/)
* [Spotify API](https://developer.spotify.com/documentation/)
* [CORS anywhere](https://cors-anywhere.herokuapp.com/)

## Deployment
* [Jammming](https://ak-playlistgenerator.surge.sh/)
